# Gasoll v3 ⛽

App de control de repostatges i gasolineres properes.

## Funcionalitats

- 🚗 Gestió de vehicles
- ⛽ Registre de repostatges amb OCR (escaneig de tiquets)
- 📍 Mapa de gasolineres properes amb preus en temps real
- 📊 Historial, estadístiques i evolució de preus
- 🔧 Control de revisions periòdiques
- 🌙 Mode fosc / clar

## Desplegar a Vercel

1. Puja aquesta carpeta a GitHub
2. Connecta el repositori a [vercel.com](https://vercel.com)
3. Vercel detecta automàticament la configuració
4. URL pública disponible en 1 minut

## Tecnologies

- HTML / CSS / JS (fitxer únic, sense dependències locals)
- [Leaflet.js](https://leafletjs.com) — Mapes
- [Tesseract.js](https://tesseract.projectnaptha.com) — OCR
- [MINETUR API](https://sedeaplicaciones.minetur.gob.es) — Preus gasolineres Espanya
